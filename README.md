## ✨ Características

- 🧠 **IA integrada** — Chat, autocomplete (FIM), Apply, Quick Edit y más con DeepSeek
- 🚀 **Rápido y liviano** — Basado en VS Code, optimizado para fluidez
- 🎨 **Tema vino tinto** — Diseño elegante con paleta de color única
- 🔧 **Configurable** — Providers locales (Ollama, vLLM, LM Studio) y cloud (DeepSeek)
- 📦 **Compatible con extensiones de VS Code**
- 🏠 **Local first** — Tus datos, tu control

## 🚀 Empezando

1. Descargá la última versión desde [releases](https://github.com/cobyzero/evi-ide/releases)
2. Abrí Evi y andá a Settings → Main Providers
3. Ingresá tu [API Key de DeepSeek](https://platform.deepseek.com/api_keys)
4. ¡Empezá a codificar con IA!

### Providers locales

También podés usar modelos locales con Ollama, vLLM o LM Studio desde la sección **Local Providers** en Settings.

## 🛠️ Desarrollo

```bash
# Clonar
git clone https://github.com/cobyzero/evi-ide.git
cd evi-ide

# Instalar dependencias
npm install

# Compilar y ejecutar
npm run watch
```
